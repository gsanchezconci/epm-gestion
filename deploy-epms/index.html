Configurar Firebase
PRIMER USO · UN SOLO VEZ
1

Crear proyecto en Firebase

Entrá a console.firebase.google.com con tu cuenta Google → Agregar proyecto → poné un nombre (ej: gestion-horas) → desactivá Google Analytics → Crear proyecto.

2

Registrar una app Web

En tu proyecto → ícono </> (Web) → poné un nombre → Registrar app. Copiá los valores del objeto firebaseConfig que aparece.

3

Activar Firestore

En el menú izquierdo → Firestore Database → Crear base de datos → elegí Modo de prueba → seleccioná la región us-central1 → Listo.

4

Pegá tu configuración acá
